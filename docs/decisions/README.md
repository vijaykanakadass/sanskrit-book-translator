# Decision Log

Every meaningful decision in this project is recorded here as a numbered file.

This gives anyone (including future AI sessions) a clear trail of **why** things are the way they are.

## Format

Each file follows this template:

```markdown
# NNN — Short Title

**Date:** YYYY-MM-DD
**Status:** Accepted | Superseded | Deprecated

## Context
What prompted this decision.

## Decision
What was decided.

## Reasoning
Why this option was chosen over alternatives.
```
